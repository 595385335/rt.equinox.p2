Provides common utility classes and services that are useful for provisioning clients. These utilities are above the level of the p2 core, but do not rely on a UI.

Package Specification

This package consists of several utility classes:

  1. LicenseManager describes a service which accepts or rejects licenses and remembers the accepted licenses. Clients should register an implementation of LicenseManager in order to remember the licenses.
  2. TranslationSupport retrieves the externalized (translated) values for installable units.
